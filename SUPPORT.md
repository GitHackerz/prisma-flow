# Support

Use the right channel so maintainers can respond quickly.

## Questions

Use GitHub Discussions for setup help, usage questions, and workflow ideas:

https://github.com/prisma-flow/prisma-flow/discussions

## Bugs

Open a bug report with reproduction steps, version details, and redacted output:

https://github.com/prisma-flow/prisma-flow/issues/new/choose

## Security

Do not open public issues for suspected vulnerabilities. Use GitHub private vulnerability reporting:

https://github.com/prisma-flow/prisma-flow/security/advisories/new

Redact database URLs, tokens, credentials, schema details, and private migration SQL from all public reports.
